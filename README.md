## Hotel and Restaurant Management System

### 📝 Description
A Java-based Hotel and Restaurant Management System. It implements complete CRUD operations using Java file handling to manage customer records, restaurant orders, and billing processes.

### 🚀 Features
- 👥 **Customer Records Management** - Create, update and view records
- 📋 **Reservation & Order Management** - Interactictive meal ordering, automatic calculation, order summary
- 💰 **Billing Module** - Reservation/Customer lookup, payment processsing, bill generation
- 📁 **File Handling** - Using .txt files to store information, temp file usage for safe record updating

### 🛠️ Technologies
- Java

### 📚 What I Learned
- File handling operations in Java (read, write, append, update)
- Working with collections to organize menu data
- Exception handling for robust applications
- Data validation and user input management
- Creating intuitive menu-driven interfaces
- Persistent data storage without databases
